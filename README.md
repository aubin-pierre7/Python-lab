# Parcours Python IA

## Objectif

Documenter une progression quotidienne en intelligence artificielle à travers des implémentations pratiques en Python : Machine Learning, Deep Learning, Vision par ordinateur, NLP et systèmes conversationnels.

## Vision

Construire une expertise solide en IA par la pratique régulière, l'expérimentation et la mise en application concrète des concepts théoriques.

## Technologies utilisées

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- TensorFlow / PyTorch
- OpenCV
- NLTK / Transformers

## Organisation du dépôt

Chaque dossier correspond à un jour d’apprentissage :

- `jour-01/`
- `jour-02/`
- `jour-03/`
- ...

Chaque dossier contient :
- Le code source
- Les expérimentations
- Les notes éventuelles

## Méthodologie

- Implémentation pratique quotidienne
- Compréhension des concepts fondamentaux
- Expérimentation et amélioration continue
- Documentation claire des apprentissages

## Objectif final

Maîtriser les bases et les techniques avancées du Machine Learning, du Deep Learning, de la Vision par ordinateur et du NLP afin de développer des applications d’intelligence artificielle robustes et performantes.
